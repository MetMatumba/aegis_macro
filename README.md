## ✨ Возможности

### 🎯 Точный движок
- **Микросекундная точность** с использованием `QueryPerformanceCounter`
- **Гибридная система тайминга** - Sleep для долгих задержек, spin-lock для микро-задержек
- **50 CPS?** Получайте 49-51 реальных кликов в секунду
- **Без дрейфа** - автоматическая синхронизация предотвращает уход времени
- **Приоритет потока** - `THREAD_PRIORITY_TIME_CRITICAL` для максимальной точности

### 👤 Управление профилями
- **Неограниченное количество профилей** с настраиваемым CPS
- **Мгновенное переключение** между профилями
- **Постоянное хранение** в базе данных SQLite3
- **Защита профиля по умолчанию**
- **Предотвращение дубликатов**

### 📊 Живая статистика
- **Счётчик реального CPS** с фактическими измерениями
- **Общий счётчик кликов** с сохранением между сессиями
- **Автосохранение** каждые 5 секунд
- **Живое обновление** интерфейса без мерцания экрана

### 🎮 Управление
| Клавиша | Действие |
|---------|----------|
| `F6` | Запуск/Остановка макроса |
| `F7` | Экстренный выход |
| `1-4` | Навигация по меню |
| `0` | Корректное завершение |

### 🔊 Звуковые сигналы
- 🟢 Запуск: Двойной сигнал (800Гц)
- 🔴 Остановка: Одиночный сигнал (600Гц)
- ✅ Успех: Восходящие тона
- ❌ Ошибка: Низкочастотный сигнал

## 🚀 Быстрый старт

### Требования
- Windows 10/11
- Visual Studio 2022 (или 2019-2026)
- Компилятор C++17-26 (MSVC)

Проект настроен для моментальной сборки в современных средах разработки.

1. **Склонируй репозиторий:**
```bash
git clone https://github.com/твое-имя/aegis-macro.git

```


2. **Открой проект:**
Запусти файл `main.slnx` через **Visual Studio 2026**.
3. **Скомпилируй и играй:**
Просто нажми **`F5`** (Запуск с отладкой) или `Ctrl + F5` в Visual Studio. Программа скомпилируется и запустится. Все зависимости (включая исходники SQLite) уже вшиты в проект.
